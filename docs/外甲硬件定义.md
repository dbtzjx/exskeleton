
# 外甲硬件定义

## 核心硬件配置

### 主控单元
- **平台**：Teensy 4.1
- **处理器**：IMXRT1062 @ 600MHz
- **控制频率**：100 Hz（10ms 周期）

### 电机与驱动系统

#### 髋关节电机（ID 1）
- **减速比**：1:36
- **控制比例**：1° = 3600 单位

#### 踝关节电机（ID 2）
- **减速比**：1:10
- **控制比例**：1° = 1000 单位
- **驱动方式**：连杆驱动

#### 通用特性
- **控制方式**：FOC 伺服电机
- **反馈**：多圈角度、速度、转矩电流、温度、故障状态

### 通讯接口
- **协议**：CAN 总线（V2.35）
- **波特率**：1 Mbps
- **帧格式**：标准帧
- **控制指令 ID**：`0x140 + 电机ID`

## 传感器与规约

### 踝关节角度定义
- **零点**：脚掌与小腿成 90°
- **正向**：背屈（向上抬脚）
- **负向**：跖屈（向下踩）

### 髋关节角度定义
- **零点**：站立姿态为零点，向前迈腿时，角度为正。
- **正向**：向前迈腿
- **负向**：向后抬腿


### 信号采集
- **相位识别**：使用髋关节角度

### 安全限位
- **踝关节**：-15°（跖屈）~ +15°（背屈）
